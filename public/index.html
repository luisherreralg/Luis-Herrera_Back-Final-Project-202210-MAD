Sneakers

  • /
    get -> all the sneakers

  • /
    post (admin) -> create a new sneaker

  • /search/:query
    get -> search for brand, model and gender (for the front filters)

  • /:id
    get -> brings a sneaker item by his id

  • /:id
    delete(admin) -> delete a sneaker by his id

  • /:id
    patch(admin) -> update the sneaker data

Users

  • /register
    post -> create a new user

  • /login
    post -> check if the user credentials are valid and returns a token

Orders

  • /cart
    get -> brings all the orders associated to the user

  • /newOrder/:itemId
    post -> create new item

  • /delete/:userId/:itemId
    delete -> Delete the item owned by the user
